# 천재교육 풀스택 과정 1차 개인 프로젝트 

# 📂 프로젝트 소개 

안녕하세요! **ITALL**은 '모든 연령'들을 위한 공부 사이트입니다.  
이 곳은 공부하고 의견을 공유하며 함께 성장하고 배우는 공간을 제공합니다.

 학생들과 교사들이 협력하고 소통하며
 **서로의 경험을 공유하고 함께 성장할 수 있는 플랫폼을 제공**하는 것이   
 저희 의도 입니다.
